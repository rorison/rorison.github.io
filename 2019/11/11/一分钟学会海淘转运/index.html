一分钟学会海淘转运
Created 2019-11-11| Updated 2019-10-20|生活
Post View:

第一步：注册中邮海外购

登陆，点击右上角的”注册”按钮进行注册：

进入注册页面后，在右侧“注册会员”界面完善注册信息(注册邮箱、密码和注册手机号)：

第二步：完善个人资料信息

成功注册后，会到达验证注册邮箱的页面，直接进入邮箱邮箱，点击中邮海外购的激活链接激活即可。接下来，便是进入转运后台的账户信息页面，填写国内收货地址等资料，并上传身份证正反面照片以便清关等。

完善个人转运信息

您可点击左侧导航的“账户中心”-> “账户信息”，按系统提示进行个人转运资料的完善(补充真实姓名、国内收件地址)等。
真实姓名默认为空，是由您自主填写的，填写中文姓名。First name 为系统自动分配的5个英文字母，为您的专属客户识别码，不可更改，仓库人员根据这客户识别码来识别您的包裹；Last Name为仓库识别码，以便海外合作的物流服务商第一时间快速、准确地将您的包裹寄达仓库。

完善国内收件人姓名和地址

点击左侧导航“账户中心”-》”我的收货地址”，即可编辑包裹国内收货人地址和收件人姓名，并上传身份证的正反两面清晰照，用于清关。注意：国内收件人姓名务必与身份证上的姓名一致，且国内收件地址需为中文。
填写好账户信息后，即可在左侧导航“我的中邮海外购”->“我的海外仓库”中，查询到您在国外网站海淘时，需填写的国外仓库专属收件地址信息（中邮海外购现已开通美中线、日中线和德中线三条转运线路，具体价格资费可查看：网站

第三步：国外海淘网站下单

在国外海淘网站下单时，务必正确填写分配给您的专属客户识别码以及海外仓地址，这样仓库人员才能帮您顺利签收入库。

第四步：创建转运预报

在国外购物网站成功下单后，您可登录中邮海外购转运后台，点击左侧的“预报包裹”创建转运预报，如下图所示：

备注：若您有多个包裹打算合箱的话，需在创建转运预报时，就勾选“需合箱”按钮）包裹预报后创建好后，直接点击“下一步”，选择您国内的收货地址提交即可。转运预报提交成功后，即可在“我的包裹管理”里查看包裹具体状态：

PS：若您为您的多个包裹均勾选“需合箱”服务，那么等您的包裹到达仓库以后，还需您选中“需合箱”的数个包裹，并点击“合箱”按钮进行合箱，以便仓库操作人员为您指定的数个包裹进行合箱操作，谢谢。

第五步：支付转运费用

转运预报提交后，坐等包裹审核通过、签收入库即可。当包裹完成称重，转运运费出来后，您再到网站上支付转运运费，然后在家坐等收货（若产生关税，美中线和德中线包裹直接到中邮海外购网站上缴纳，中邮海外购会帮您代缴给海关）。
